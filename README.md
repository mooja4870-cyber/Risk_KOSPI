# Risk_KOSPI

## Streamlit deploy

This repository is prepared for Streamlit Community Cloud deployment.

### Entry point

- `app.py`

### Local run

```bash
pip install -r requirements.txt
streamlit run app.py
```

### Community Cloud

1. Push this repo to GitHub.
2. Open Streamlit Community Cloud and create a new app from this repo.
3. Set branch to `main` and main file path to `app.py`.
